LongText Databasse Activity field for Moodle
============================

A data field type component to moodle. Write long text controlling the max chars.

Package tested in: moodle 3.7+

QUICK INSTALL
==============
Download zip package, extract the longtext folder and upload this folder into mod/data/field/.
